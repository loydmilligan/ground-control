# Requirements: 5.1 Create FocusedFeed Component

## Description

Create FocusedFeed.tsx as THIN WRAPPER around existing NotificationFeed with mode=focus. Do NOT create new component - reuse NotificationFeed.tsx patterns. Accept notifications and onAction props.

## Requirements

- Shows only notifications passing all filters
- No RuleBar or filter controls

- Integrates as Stage 3 in carousel

## Constraints

- Dark theme: bg-slate-900 base
- Min 44px touch targets
